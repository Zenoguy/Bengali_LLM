> **"বাংলার জন্য তৈরি। অন্য ভাষা থেকে অনুবাদ নয়।"**  
> (Built for Bengal. Not just a translation from other languages.)

Developed by the **CU Data Science Lab**, this project represents a significant step towards culturally grounded and linguistically accurate Large Language Models for the Bengali language. Unlike standard models that treat Bengali as a secondary translation task, our model is instruction-tuned on authentic Bengali datasets to capture the nuances, idioms, and context of the soil.

---

## ✨ Key Features

- 🧠 **Instruction-Tuned Intelligence**: Fine-tuned on **BanglaRQA** and proprietary instruction datasets covering real-world Bengali scenarios.
- ⚡ **LoRA Optimized**: Leverages Low-Rank Adaptation (LoRA) and 4-bit quantization for high-performance inference even on consumer hardware.
- 💬 **Premium Immersive UI**: A state-of-the-art "Maati & Shorshe" (Terracotta & Mustard) themed interface designed for maximum cultural immersion.
- 📂 **Dynamic Session Management**: Persistent chat history with auto-naming based on conversation context.
- 🌐 **Localized Experience**: Fully localized interface supporting both Bengali and English seamlessly.

---

## 🎨 Design Philosophy: "Maati & Shorshe"

The interface is built on a custom design system that honors Bengal's heritage:
- **Kora Off-White (`#F5F5DC`)**: Inspired by unbleached cotton (Kora) fabric.
- **Terracotta Red (`#B22222`)**: Reflecting the historic burnt-clay temples of Bishnupur.
- **Mustard Yellow (`#FFDB58`)**: Inspired by the vast Shorshe (Mustard) fields of Bengal.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Zenoguy/Bengali_LLM.git
   cd Bengali_LLM/project
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 📊 Evaluation Metrics

Our model is rigorously tested across multiple dimensions:
- **F1 Score**: For factual accuracy.
- **ROUGE/METEOR**: For linguistic fluency and structural integrity.
- **Human Eval**: For cultural and contextual nuance.
